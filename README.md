# Game Of Life

---

This is my first C++ project, a recreation of Conway's Game Of Life

This project was written on NixOS using [dev-templates'](https://github.com/the-nix-way/dev-templates) C/CPP Nix Flake template, with raylib and pkg-config added


## Building

---

#### Requirements:
- C++20
- g++
- Make
- pkg-config
- [RayLib](https://github.com/raysan5/raylib) + [RayGui](https://github.com/raysan5/raygui)
- (optional) Bear - for ```compile_command.json``` generation
- (optional) Nix - for using the ```flake.nix``` environment, which installs all dependencies
